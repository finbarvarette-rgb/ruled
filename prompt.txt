PART 3 — COURT PREP EXPERIENCE  
Create app/dashboard/cases/[id]/court-prep/page.tsx

This page shows the 5-step sequential court prep flow.
Read the case data and full case pack content from 
the database.

Show 5 step cards stacked vertically:

Step 1 — Review demand letter
Status: always done if case pack purchased
Action link: "View Letter" → /dashboard/cases/[id]/demand-letter

Step 2 — Confirm intent to file
If not confirmed: show button "Yes, I want to file in court"
  Clicking saves confirmed_filing=true to the case in Supabase
If confirmed: show green checkmark, "Confirmed"

Step 3 — File your claim
Show filing instructions inline when expanded:
- Province-specific courthouse name and address
- Filing fee amount for their claim size
- Download claim form button (downloads the How to File PDF)
- "I've Filed My Claim" confirmation button
  Clicking saves filed=true to the case in Supabase
- Note: do not auto-mark this as filed, user must confirm

Step 4 — Serve the defendant
Show service instructions inline when Step 3 is confirmed:
- Three methods: personal service, registered mail, email
- Download Proof of Service form button
- "I've Served the Defendant" confirmation button

Step 5 — Prepare for your hearing
Unlocks after Step 4. Shows 4 tabs:
Tab 1 — Opening Script:
  Pull the opening script from the case pack data in 
  the database. Display in a styled script card.
  Download button for this section.
Tab 2 — Closing Script:
  Same, pull from database.
Tab 3 — Defence Arguments:
  Pull anticipated defence arguments from database.
  Each argument shown as an amber-titled card with 
  the suggested response below it.
Tab 4 — Day Of Checklist:
  Static checklist of items to bring and do.
  Checkboxes that save state to localStorage.
  Items: printed documents x3, photo ID, scripts, 
  arrive 30 min early, address judge as Your Honour,
  speak clearly and wait your turn.
